# Telegram Giveaway Bot

Бот для розыгрышей в Telegram-канале с веб-интерфейсом управления.

## Что умеет

- Создавать проекты в админке (название, лого, реф-ссылка).
- Создавать розыгрыши через веб-панель с привязкой к проекту.
- Прикреплять изображение к посту розыгрыша.
- Публиковать сразу или планировать по времени.
- Завершать вручную в любой момент или по расписанию.
- Добавлять кнопку участия `Участвовать (N)` и считать участников.
- Автоматически выбирать победителей при завершении.
- Скрывать список победителей в посте (без публикации ID).
- Запоминать для пользователя по проекту: подтверждение реферала и TRC-20 адрес.

## 1) Установка

```bash
npm install
```

## 2) Настройка

Скопируйте `.env.example` в `.env`:

```bash
cp .env.example .env
```

Заполните переменные:

- `BOT_TOKEN` — токен от @BotFather
- `ADMIN_IDS` — Telegram ID админов через запятую (например `123,456`)
- `TIMEZONE` — часовой пояс для дат (например `Europe/Moscow`)
- `CHECK_INTERVAL_MS` — как часто проверять запланированные задачи
- `WEB_PORT` — порт веб-панели (например `3000`)

## 3) Права бота в канале

1. Добавьте бота администратором в канал.
2. Дайте права на публикацию и редактирование сообщений.
3. В веб-панели указывайте `@username` канала или ID канала (`-100...`).

## 4) Запуск

```bash
npm run start
```

После запуска:

- бот работает в Telegram;
- веб-панель доступна по адресу: `http://localhost:3000` (или ваш `WEB_PORT`).

## 5) Как создать розыгрыш через веб

1. Откройте `http://localhost:3000`.
2. Сначала добавьте проект (название, реф-ссылка, лого).
3. Заполните форму розыгрыша:
   - проект,
   - канал (из списка подключенных или вручную),
   - приз,
   - описание,
   - картинка (опционально),
   - количество победителей,
   - режим публикации: сразу или по времени,
   - режим завершения: вручную или по времени.
4. Нажмите **Создать**.

В списке розыгрышей доступны кнопки:

- **Опубликовать сейчас** (для запланированных);
- **Завершить сейчас** (для активных).

## Авто-определение каналов

Бот автоматически запоминает каналы, в которых он присутствует:

1. Добавьте бота администратором в канал.
2. Опубликуйте любой пост в канале.
3. Канал появится в выпадающем списке веб-панели и в команде `/channels`.

## 7) Этапы участия пользователя

При нажатии кнопки `Участвовать` пользователь проходит этапы в личном чате с ботом:

1. Простая проверка "Я не робот" (арифметический вопрос).
2. Подтверждение перехода по реферальной ссылке проекта.
3. Ввод TRC-20 адреса.

После первого успешного прохождения для конкретного проекта:

- реферальный статус и TRC-20 адрес сохраняются;
- в следующих розыгрышах этого же проекта этапы повторно не запрашиваются.

## 6) Команды Telegram (дополнительно)

- `/start` — справка
- `/create_draw` — создание розыгрыша через чат
- `/my_draws` — последние розыгрыши
- `/cancel_draw` — отменить текущее создание
